# 听画

听画是一款本地运行、通过简体中文语音控制的结构化矢量绘图工具。首次启动和麦克风授权可以使用鼠标；进入画板后，创建、编辑、布局、撤销、清空与导出均可通过语音完成。

## 启动

要求 Python 3.8+，建议使用 Chrome 或 Edge。云端模式需要浏览器支持 `MediaRecorder`、`AudioContext`；浏览器模式需要 Web Speech API。

```powershell
python main.py
```

服务默认打开 `http://127.0.0.1:8000`。点击一次“开始聆听”会直接启动当前识别模式并请求麦克风权限；完整聆听中说“休息”可进入后台唤醒词模式。若不希望自动打开浏览器：

```powershell
python main.py --no-browser
```

服务会独占监听端口，避免多个旧进程同时响应请求导致 `/api/transcribe` 随机返回 404。若提示端口已占用，请先关闭旧的听画服务。

云端语音转写和浏览器识别通常依赖网络。未配置 `OPENAI_API_KEY` 时，页面会自动切换到浏览器识别并显示原因。应用本地规则解析、绘图和导出不需要额外依赖。

浏览器模式会优先使用绘图领域词汇偏置和多个识别候选；稳定且能由本地规则确定解析的中间结果会在约 400ms 后直接执行，以减少等待最终识别结果的延迟。

## 离线语音识别（可选）

听画支持通过 Vosk.js WASM 在无网络环境下进行中文语音识别。离线模式作为可选增强，默认仍优先使用云端识别。

### 启用方式

1. 启动应用后，点击 UI 中的 **"切换模式"** 按钮可切换可用的识别模式；离线模型就绪后才会加入切换循环：
   - 💻 离线：使用 Vosk WASM 本地识别，无需网络
   - 🌐 云端：使用 OpenAI STT 云端转写（默认，需配置 API）
   - 🌐↓ 浏览器：使用浏览器 Web Speech API 降级识别

2. 首次使用离线模式需下载中文模型（约 42MB）：
   - 点击 **"下载离线模型"** 按钮自动下载并缓存到浏览器 IndexedDB
   - 或参考 `static/vosk/README.md` 手动配置本地模型文件

3. 模型下载完成后，切换至离线模式即可断网使用

### 降级链

应用按以下优先级自动选择识别模式：离线（模型已缓存）→ 云端（网络可用）→ 浏览器（降级）

### 限制

- 离线模型准确率低于云端，建议配合模糊匹配使用
- 模型较大（42MB），首次下载需等待
- 清除浏览器数据会导致模型丢失，需重新下载

## 云端高准确率语音识别

应用默认使用云端音频转写，并通过静音检测自动切分每条指令。配置 OpenAI 兼容的音频转写接口后重启服务：

```powershell
# 在 .env 中填写 OPENAI_API_KEY
# 在 config.yaml 中配置服务地址和模型
python main.py
```

服务启动时自动读取项目根目录的 `.env` 和 `config.yaml`。`.env` 只保存 `OPENAI_API_KEY` 且已被 Git 忽略，可提交的密钥模板为 `.env.example`；服务地址和模型名均写在可提交的 `config.yaml` 中。音频只转发到配置的转写服务，不会保存到本地磁盘。

当前示例配置使用阿里云百炼兼容接口：语音转写模型为 `qwen3-asr-flash`，命令模型为 `qwen-plus`。实时 WebSocket 模型（名称通常包含 `realtime`）不能用于当前短音频转写请求。

云端请求失败时，页面会区分缺少密钥、密钥无效、权限不足、额度不足、模型或接口不存在、网络错误和连接超时。

## 示例指令

- `画一个红色圆形放在左边，再画一个蓝色矩形放在右边`
- `画一个文字，写上“你好”放在中央`
- `选中刚才两个图形，然后顶部对齐，然后整体向下移动五十`
- `选择矩形一，然后把它改成绿色`
- `把它放大两倍，然后旋转四十五`
- `选择所有图形，然后横向均匀分布`
- `撤销`、`重做`、`当前状态`、`帮助`
- `设置背景为灰色`
- `清空画布`，随后说 `确认` 或 `取消`
- `保存为 SVG`、`保存为 PNG`

每个图形会按类型自动命名，例如“矩形一”“矩形二”“圆形一”。

## 可选模型回退

规则解析器不能理解指令时，服务端可调用 OpenAI 兼容的聊天补全接口。接口地址和模型通过 `config.yaml` 的 `command_model` 配置。

密钥只保留在 Python 服务端。模型返回的动作会在服务端和浏览器内校验，每条语音最多执行 20 个动作。未配置模型时，全部标准指令仍可使用。
